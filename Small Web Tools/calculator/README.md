# Calculator Application

This project is a simple calculator application that performs basic arithmetic operations such as addition, subtraction, multiplication, and division. 

## Files Overview

- **index.html**: The main HTML document that structures the user interface of the calculator, including input fields, buttons, and a display area for results.

- **style.css**: Contains the styles for the calculator application, defining the layout, colors, fonts, and other visual aspects to enhance the user interface.

- **app.js**: The JavaScript code that implements the calculator's functionality. It exports functions for basic arithmetic operations and handles user interactions, such as button clicks and input processing.

## Instructions for Use

1. Open `index.html` in a web browser to access the calculator.
2. Use the buttons to input numbers and select operations.
3. The result will be displayed in the designated area after performing calculations.

## Additional Information

This project is intended for educational purposes and can be expanded with more features, such as advanced mathematical functions or a history of calculations.
